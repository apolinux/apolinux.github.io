Cime

Control de Citas Médicas

Es una aplicación que permite la administración de citas médicas. Sirve para manejar un pequeño centro médico o puede ser un módulo para un proyecto mayor. Puede realizar administración de pacientes, médicos, horarios, permisos de asistente, administrador y usuario normal. Es hecho para trabajar totalmente desde la web. Tiene una interfaz de usuario bastante fácil de manejar e intuitiva. Como se usa desde la web, es posible trabajar offline u online.

Puedes ver más información sobre esta aplicación en Github.
